# 스타일 프리셋 레퍼런스

7종 블로그 글쓰기 스타일 프리셋. 리서치 기반 구조 유형 9종 + 톤 유형 6종을 조합하여 설계.
스타일 매칭 엔진(`src/style_matcher.py`)이 주제 특성에 맞는 프리셋을 자동 선택한다.

---

## 1. info_delivery (정보전달형)

**구조**: 화제 소개 → 핵심 정보 정리 → 왜 중요한지 → 실생활 적용 → 마무리
**톤**: 편안한 존댓말, 리포터처럼 명확하게. "빠르게 핵심만 알려드릴게요" 느낌.
**길이**: 2000-2500자
**이미지 전략**: 트렌드 제품/서비스 스크린샷, 사용 장면. 현대적이고 세련된 스타일.
**매칭 시그널**: 소개, 정보, 알려줘, 뉴스, 요약, 트렌드, 유행, 화제, 요즘, 최신
**카테고리 친화도**: ai_tech(0.9), health(0.6), self_improvement(0.5)
**사용 사례**: "요즘 AI 코딩 도구가 화제야", "ChatGPT 새 기능 나왔다", "두쫀쿠 유행한다던데"

---

## 2. review_experience (리뷰 체험형)

**구조**: 제품/서비스 소개 → 직접 써본 경험 → 장점 → 단점/아쉬운 점 → 비교 → 종합 평가
**톤**: 솔직하고 객관적, 직접 해본 사람의 신뢰감. "마케팅 아니고 진짜 후기" 느낌.
**길이**: 2500-3000자
**이미지 전략**: 실제 제품/서비스 스크린샷, 비교 테이블, 별점 시각화.
**매칭 시그널**: 리뷰, 후기, 써봤, 해봤, 체험, 비교, vs, 장단점, 평가
**카테고리 친화도**: ai_tech(0.8), health(0.7), self_improvement(0.4)
**사용 사례**: "ChatGPT vs Claude 비교", "갤럭시 워치 한 달 사용기", "필라테스 3개월 후기"

## 3. essay_emotional (에세이 감성형)

**구조**: 에피소드/장면 → 느낀 점 → 생각의 확장 → 깨달음 → 여운 있는 마무리
**톤**: 따뜻하고 사색적, 에세이스트의 감성. "같이 생각해봐요" 느낌.
**길이**: 2000-2500자
**이미지 전략**: 감성적 일러스트, 따뜻한 색감, 부드러운 톤. 소수 감성 이미지.
**매칭 시그널**: 에세이, 감성, 느낀, 생각, 마음, 변화, 도전, 성장, 회고, 깨달음
**카테고리 친화도**: ai_tech(0.2), health(0.7), self_improvement(0.9)
**사용 사례**: "퇴사 후 100일 회고", "아침 루틴이 바꾼 것들", "30대가 되어 깨달은 것"

## 4. list_curation (리스트 큐레이션형)

**구조**: 도입 (왜 이 리스트인지) → 항목별 소개(5-10개) → 비교/요약표 → 최종 추천
**톤**: 가볍고 읽기 쉬운 큐레이터. "이것만 보면 됩니다" 느낌.
**길이**: 2000-3000자 (항목당 200-300자)
**이미지 전략**: 각 항목 대표 이미지 또는 통합 인포그래픽. 밝고 컬러풀한 스타일.
**매칭 시그널**: 추천, 모음, 베스트, 리스트, top, best, 선정, 정리, 골라봤
**카테고리 친화도**: ai_tech(0.6), health(0.8), self_improvement(0.7)
**사용 사례**: "제주도 카페 추천 10선", "AI 도구 베스트 5", "직장인 건강 습관 7가지"

## 5. expert_analysis (전문가 분석형)

**구조**: 화제 소개 → 배경 설명 → 데이터/근거 기반 분석 → 시사점 → 전망/예측

**길이**: 2500-3000자 (데이터/인용 포함)
**이미지 전략**: 데이터 시각화, 차트, 타임라인 스타일. 깔끔하고 전문적인 톤.
**매칭 시그널**: 분석, 이유, 왜, 영향, 의미, 전망, 데이터, 통계, 정책, 경제
**카테고리 친화도**: ai_tech(0.8), health(0.5), self_improvement(0.3)
**사용 사례**: "이란 미사일과 기름값의 관계", "AI 반도체 시장 전망", "정부 정책 해설"

## 6. friendly_chat (친근한 대화형)

**구조**: 상황 공감 → 핵심 정보/꿀팁 → 왜 이게 좋은지 → 주의할 점 → 친근한 마무리
**톤**: 친구처럼 편안한 대화체. 반말+존댓말 믹스. "야 이거 진짜 좋더라" 느낌.
**길이**: 1500-2500자 (짧고 임팩트 있게)
**이미지 전략**: 캐주얼한 밈/캡처, 일상 사진 스타일. 가벼운 분위기.
**매칭 시그널**: 꿀팁, 알려줄게, 진짜, 대화, 편하게, 쉽게, 간단, 빠르게
**카테고리 친화도**: ai_tech(0.4), health(0.8), self_improvement(0.6)
**사용 사례**: "직장인 점심 꿀팁", "이거 안 쓰면 손해", "다이어트 할 때 이것만은"

## 7. tutorial_howto (튜토리얼/하우투형)

**구조**: 뭘 만들/할 건지 → 준비물/사전조건 → 단계별 실행(Step 1-N) → 결과 확인 → 트러블슈팅/팁
**톤**: 친절한 선생님, 명확하고 정확하게. "따라만 하면 됩니다" 느낌.
**길이**: 2500-3000자 (단계 설명이 길어질 수 있음)
**이미지 전략**: 각 단계별 스크린샷/다이어그램 스타일. 번호가 매겨진 시각적 가이드.
**매칭 시그널**: 방법, 하는법, 설치, 설정, 만들기, 따라하기, 가이드, how to, tutorial
**카테고리 친화도**: ai_tech(0.9), health(0.4), self_improvement(0.5)
**사용 사례**: "파이썬 설치 방법", "노션 활용 가이드", "맥북 초기 설정"

## 스타일 매칭 공식

```
final_score = (keyword_score * 0.2) + (llm_score * 0.6) + (category_score * 0.2)
```

- **keyword_score**: TopicSpec.keywords와 프리셋 match_signals의 겹침 비율 (0~1)
- **llm_score**: Claude가 주제의 intent/complexity를 분석하여 판단한 적합도 (0~1)
- **category_score**: 프리셋의 카테고리 친화도 값 (0~1)

최고점 프리셋을 추천하되, 0.1 이내 차이면 2개를 함께 제시한다.
